# Zen Focus 🎯

O **Zen Focus ** é uma aplicação Full-Stack focada em produtividade, projetada para oferecer uma experiência de usuário (UX) fluida, moderna e eficiente. O sistema permite o gerenciamento completo de tarefas, categorização, definição de prioridades e acompanhamento de métricas em tempo real.

## 🚀 Funcionalidades

- **Dashboard de Produtividade:** Métricas claras de tarefas concluídas e pendentes.
- **Gestão de Tarefas:** Adicione, categorize e priorize suas atividades.
- **Interface Moderna:** Design responsivo com suporte a Dark Mode.
- **PWA (Progressive Web App):** 📱 Instale o Zen Focus como um aplicativo nativo no seu **celular** ou **PC** através do navegador e acesse suas tarefas com um clique!

## 🛠️ Tecnologias Utilizadas

- **Frontend:** React.js / Vite
- **Backend:** Node.js
- **Banco de Dados:** Firebase
- **Arquitetura:** PWA (Progressive Web App)
